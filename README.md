# Codebook
My own Codebook for cp

## 1. How to Build
**Step 1:** Go to the project root and build by
~~~
$ cd Codebook
$ python GenCodebookList.py
$ xelatex Codebook.tex && xelatex Codebook.tex
~~~

### 1.1. Dependencies

* python (version >= 3.0)
* xelatex (latest version)
